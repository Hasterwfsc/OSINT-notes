# Phone Number OSINT

## Phoneinfoga

```
phoneinfoga scan -n 14082492815
phoneinfoga scan -n <number>
```



{% hint style="info" %}
We need to put our country code in front of the number

for the example above 1 is the country code for the US&#x20;
{% endhint %}
