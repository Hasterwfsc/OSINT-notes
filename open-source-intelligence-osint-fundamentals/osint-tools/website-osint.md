### Subfinder - [https://github.com/projectdiscovery/subfinder](https://github.com/projectdiscovery/subfinder)

#### To install

```
go install -v github.com/projectdiscovery/subfinder/v2/cmd/subfinder@latest
```

#### Example usage&#x20;

```
subfinder -d tcm-sec.com 
```
